# 🍁 MappleFlix

> A Netflix-inspired streaming site built in a **single HTML file** — powered by the TMDB API and VidSrc for free embedded playback.

![MappleFlix Banner](https://image.tmdb.org/t/p/original/628Dep6AxEtDxjZoGP78TsOxYbK.jpg)

---

## 🚀 Live Demo

Just open `index.html` in any modern browser — no build tools, no install, no server required.

---

## ✨ Features

### 🎨 Netflix-Style UI
- Deep `#141414` background with iconic **red (`#E50914`)** accent color
- Fixed navbar with **scroll-aware transition** — gradient at top, solid when scrolled
- **Animated hero banner** that auto-rotates featured content every 7 seconds
- Smooth card **hover zoom + overlay** animations
- Custom **red scrollbar** styling
- Fully **responsive** — works on mobile, tablet, and desktop

### 🗂️ Content Tabs

| Tab | Sections |
|-----|----------|
| 🎬 **Movies** | Trending, Popular, Top Rated, Upcoming, Action, Horror, Comedy, Sci-Fi |
| 📺 **TV Shows** | Trending, Popular, Top Rated, Currently Airing, Drama, Crime |
| 🔥 **Trending** | All Trending, Trending Movies Today, Trending TV Today |
| ⚡ **Anime** | Popular Anime, Top Rated Anime, Latest Anime |

### 🔍 Live Search
- Real-time search with a **400ms debounce** to avoid excessive API calls
- Results appear in a full **overlay panel** above the content
- Covers both Movies and TV Shows simultaneously

### ▶️ Player Modal
- Click any card to open a **detailed info modal**
- Shows title, rating, release year, runtime/seasons, genres, and full overview
- **Play button** loads the stream via `vidsrc.to` in a responsive iframe
- **TV Shows** include a full **Season Selector** and **Episode Grid** — click any episode to stream directly
- Close with the ✕ button, **ESC key**, or clicking the backdrop

### 🔔 Toast Notifications
- Brief pop-up notifications for player load events and errors

## 📡 APIs Used

### 1. TMDB (The Movie Database)
- **Base URL:** `https://api.themoviedb.org/3`
- **Image CDN:** `https://image.tmdb.org/t/p/`
- **Used for:** Fetching movie/TV metadata, posters, backdrops, genres, episode lists, search
- **Docs:** [https://developers.themoviedb.org/3](https://developers.themoviedb.org/3)

> ⚠️ The project uses a **public demo API key**. For production use, register your own free key at [themoviedb.org](https://www.themoviedb.org/settings/api).

### 2. VidSrc.to
- **Embed URL format:**
  - Movie: `https://vidsrc.to/embed/movie/{tmdb_id}`
  - TV Episode: `https://vidsrc.to/embed/tv/{tmdb_id}/{season}/{episode}`
- **Used for:** Free, embeddable streaming player using TMDB IDs
- **Docs:** [https://vidsrc.to](https://vidsrc.to)

## 📁 Project Structure

```
mappleflix/
└── index.html       # The entire application — HTML + CSS + JS
└── README.md        # This file
```

Everything lives in `index.html`:

```
index.html
├── <head>
│   ├── Tailwind CSS CDN
│   ├── Google Fonts
│   └── <style> — All custom CSS
├── <body>
│   ├── <nav>          — Fixed navbar with tabs + search
│   ├── #searchResults — Live search overlay
│   ├── #hero          — Auto-rotating hero banner
│   ├── <main>         — Dynamic content rows (generated by JS)
│   ├── #modal         — Video player + info modal
│   ├── #toast         — Notification system
│   └── <footer>
└── <script>           — All JavaScript logic
    ├── CONFIG         — API keys and base URLs
    ├── fetchTMDB()    — Reusable TMDB fetch helper
    ├── loadPage()     — Renders rows for current tab
    ├── startHero()    — Hero banner rotation logic
    ├── renderRow()    — Builds horizontal scroll rows
    ├── openModal()    — Fetches details + opens modal
    ├── loadPlayer()   — Injects VidSrc iframe
    ├── handleSearch() — Debounced live search
    └── switchTab()    — Tab switching
```

## ⚙️ Configuration

Open `index.html` and find the config block near the top of the `<script>` tag:

```javascript
// ===== CONFIG =====
const TMDB_KEY = 'your_tmdb_api_key_here';
const TMDB    = 'https://api.themoviedb.org/3';
const IMG     = 'https://image.tmdb.org/t/p/';
const VIDSRC  = 'https://vidsrc.to/embed';
```

Replace `TMDB_KEY` with your own key from [https://www.themoviedb.org/settings/api](https://www.themoviedb.org/settings/api).

## 🎬 How Streaming Works

1. User clicks a movie/TV card
2. `openModal()` fetches full details from TMDB
3. User clicks **▶ Play**
4. `loadPlayer()` constructs a VidSrc URL using the TMDB ID
5. An `<iframe>` is injected into the modal's player wrapper
6. For TV shows — season buttons fetch episode lists; clicking an episode streams that specific episode

```
Movie stream URL:
  https://vidsrc.to/embed/movie/{tmdb_id}

TV episode stream URL:
  https://vidsrc.to/embed/tv/{tmdb_id}/{season_number}/{episode_number}
```
